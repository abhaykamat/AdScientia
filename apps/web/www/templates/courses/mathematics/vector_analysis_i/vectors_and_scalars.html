Vector Spaces, Scalar Fields and Vector Fields

Vector Spaces

Definition
A $\textbf{vector space}$ is a set $V$ over a field $\mathbb{F}$ s.t. the following axioms hold

For the entirety of this course we will take our field $\mathbb{F}$ to be the set of real numbers, $\mathbb{R}$.

Scalar Fields

Definition
Let $D \subset \mathbb{R}^n$, then $f:D \rightarrow \mathbb{R}$ is a $\textbf{scalar field}$.

% Example 1 - Temperature Field in a room
An example of a scalar field is given by the temperature field $T(\textbf{x})$ at a point $\textbf{x} \in D$, in a room $D$, given by $T:D \subset \mathbb{R}^3 \rightarrow \mathbb{R}$.

% Example 2 - Height Field on a cartographic map
Another example is given by the height field $H(\textbf{y})$ at a point $\textbf{y} \in D$, of a mountain on a cartographic map $D$, given by $H:D \subset \mathbb{R}^2 \rightarrow \mathbb{R}$.

Vector Fields

Definition
Let $D \subset \mathbb{R}^n$, then $f:D \rightarrow \mathbb{R}^m$, $m>1$ is a $\textbf{vector field}$.

% Example 1 - Velocity Field of a river
An example of a vector field is given by the velocity field $\textbf{U}(\textbf{x})$ at a point $\textbf{x} \in D$, in a river $D$, given by $\textbf{U}:D \subset \mathbb{R}^3 \rightarrow \mathbb{R}$.

% Example 2 - Gravitational Field
Another example is given by the gravitational force field $\textbf{g}(\textbf{r})$ acting on a particle at a point $\textbf{r} \in D$, given by $\textbf{g}:D \subset \mathbb{R}^3 \rightarrow \mathbb{R}$.
